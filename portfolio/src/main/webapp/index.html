Projects:

Timelines: Timelines is mobile app that lets busy people find free time to spend with their loved ones. Input your schedule, add someone as a friend, and then sync your schedule with theirs to find shared free time.

Maze Game: A 3D Maze Game built from scratch in Java that can aptly be described as first-person Pac-Man. Collect all the gold coins in the the maze to win!

InTheDark: A text based game where you explore a perilous underground cave system. Defeating gruesome and grotesque monsters and collecting their loot while slowly ascending back to the surface.

Polynomial Calculator: A terminal application built in C++ that allows you to enter as many polynomials as you want and then perform addition or multiplication on them.
